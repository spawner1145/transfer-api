本地启动

需要 Node.js 18+。现在可以直接启动；如果没有配置 UNLIMITED_SURF_API_KEY，后台会先自动请求 https://unlimited.surf/api/key 获取 key，并且每小时自动刷新一次。网页端会自动使用当前站点地址，不需要手动填写 Base URL。

# Windows PowerShell
# 可选：只限制浏览器/客户端调用本代理时使用的 key
$env:WORKER_API_KEY = "<可选：客户端 key>"
node server.js

默认访问：http://localhost:6008。如果设置了 WORKER_API_KEY，进入网页 UI 前会弹窗要求输入，验证成功后保存在本地 Cookie。

OpenAI 兼容调用

curl http://localhost:6008/v1/chat/completions \
  -H "Authorization: Bearer <你的客户端 key；未设置 WORKER_API_KEY 时可填任意值>" \
  -H "Content-Type: application/json" \
  -d '{"model":"gateway-gpt-5-5","messages":[{"role":"user","content":"Hello"}],"stream":true}'

Anthropic / Claude Code

$env:ANTHROPIC_BASE_URL = "http://localhost:6008"
$env:ANTHROPIC_AUTH_TOKEN = "<你的客户端 key；未设置 WORKER_API_KEY 时可填任意值>"
$env:ANTHROPIC_API_KEY = "<你的客户端 key；未设置 WORKER_API_KEY 时可填任意值>"
$env:ANTHROPIC_MODEL = "claude-opus-4-7-20260101"

常用端点

  • GET /health：服务状态
  • GET /v1/models：模型列表
  • POST /v1/chat/completions：OpenAI 聊天
  • POST /v1/messages：Anthropic 聊天
  • POST /v1/search、POST /v1/merge：搜索与多模型合并能力
